## Решения

- **location_developers переделана под leaf-уровень** (в параллельном чате). Project-блок матчится
  по `location_id = property.location_id`, `developer_name` денормализован (JOIN developers не нужен).
  Маппинг: Name←project_group_name; Building/Cluster/Project по `is_building`; Completion из
  project_status (under_construction/planned→Off-Plan, completed→Ready); Handover completed→built_year,
  иначе completion_q+year. Наполнено пока только Damac Hills — по другим районам project=null (норма).
- **Vastu** — только residential apartment/house; в ленте НЕ показываем (число beds + maid серым).
- **Реализацию веду через Subagent-Driven** (пользователь поправил после прямой правки ленты) — даже
  мелкие правки субагентами. Сохранено в память.
- **get_property переписывали один раз** после rework (active_listings/slider/is_vastu/project вместе),
  чтобы не делать двойную работу.

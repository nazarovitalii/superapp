## Решения

- **Чекбокс в ленте = множественный выбор** (не «отработано» и не избранное) —
  задел под массовые действия: сравнить, отправить клиенту, создать задачи.
- Hover-кнопки показываются только на устройствах с мышью (`@media (hover: hover)`).
- Везде используются готовые токены SP (`--task-detail-bg`, `--task-c-selected-bg`,
  `--task-shadow`…) — все темы (тёмная/светлая/кастомные) работают автоматически.

### Что сделали

- **Активная строка** (карточка открыта) — убрали цветовую подсветку, вместо этого тень
  `--task-selected-shadow` + box продлевается вправо до sidebar. Точная копия `.isSelected`
  из `_task-base.scss` (те же `right: -34px / -234px` по брейкпоинтам).
- **Новая колонка 📷** — фотоаппарат появляется, если у листинга есть фото. Без заголовка.
- **Агентство + дата** вместо Pocket/Official: имя агентства + `last_actualized_at || published_at`.
- **Сортировка в хедере** — меню `swap_vert`: Сначала новые / дорогие / дешёвые / давние → `p_sort_by`.
- **Охват в хедере** — пилюля-меню: Все / Мои объекты / Объекты сети / Public. Клиентский фильтр
  (нет `p_scope` в get_feed — записано как API-9).
- **Фильтры переписаны 1:1 с реальным `get_feed`**: типы недвижимости из справочника БД,
  спальни/санузлы — мультиселект (массивы), цена с AED и запятыми, площадь sqft, мебель, готовность,
  листинг чипами. Убран «Только срочные» (нет в API).
- **Индикация фильтров** — при ≥1 активном фильтре `tune` → синий чип «tune N».
- **Хедер очищен**: убраны Today-плашка с точками, Play, Focus Mode, Sync, кнопка провайдеров.
- **Форма добавления объекта**: вернули шаги с нумерацией (кружки 1–5 сверху), блок крупнее
  (max-width 680px, шрифты 15–17px, чипы padding 8×16px).
- Тесты (6/6 ✅), checkFile чистый, билд чистый.

### Решения

- **Охват — клиентский** (owner_id / is_network / visibility), потому что в `get_feed` нет
  серверного параметра. count_total и пагинация не учитывают охват — записано как задача API-9.
  Исправить можно миграцией RPC (с разрешения).
- **Форма = визард** (а не одна страница блоками) — пользователь вернул шаги после первого
  фидбека «уродство». Блок и шрифты стали крупнее (17px vs 14px).
- Pipe precedence Angular: `a || b | date` → `(a || b) | date` — исправлено.

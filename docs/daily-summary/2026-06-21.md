## Гочи (на память)

- **Postgres ARE: `\b` = backspace, НЕ граница слова** (→ `\y`/`\m` или якорь на пунктуацию). Из-за `\b` патч `get_property` сначала молча не совпал; staleness-proof DO-блоки должны иметь guard `IF v_new <> v_def … ELSE RAISE NOTICE`.
- `get_property` с `auth.uid()`: для проверки из CLI передавать `p_user_id`=owner (COALESCE).

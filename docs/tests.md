# Журнал тестов — MrSQM (superapp)

Фиксируем **успешные прод-тесты** с кодом запроса, ожидаемым и фактическим результатом.

---

## Рутина

После каждого прод-теста добавляй запись:
```
### T-<N>: <что проверяли>
**Дата:** YYYY-MM-DD
**Триггер:** <что изменилось / зачем тест>
**Запрос:** <curl / суть>
**Ожидали:** ...
**Получили:** ...
**Вывод:** ...
```
